# Reporting Issues

- Post at **Issues** tab.
- Standard format:

Title:
```
[Tag] Short descriptive title
```

Content:
```
Release version
Python version
Short Summary
Error message
Screenshot (if applicable)
```

### Tags allowed:
- BUG
  - Bugs or errors.
- FEATURE REQUEST
  - Request new feature.
- CHANGE
  - Behavioural change of feature.
